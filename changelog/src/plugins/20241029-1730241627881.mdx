### October 29, 2024
## extractors

@flatfile/plugin-pdf-extractor@0.4.0
The release updates the PDF extractor plugin to version 0.4.0. It swaps the package bundler to tsup, which is a newer and faster bundler. The plugin now has separate builds for Node.js and browser environments. The package exports have been updated to support multiple entry points for different environments. The build scripts have been updated to use tsup instead of Parcel. New lint and check scripts have been added for better code quality. The util-file-buffer and util-common dependencies have been updated to their latest versions. The package.json file has been updated with additional metadata fields.
