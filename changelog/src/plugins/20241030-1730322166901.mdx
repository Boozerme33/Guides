### October 30, 2024
## extractors

@flatfile/plugin-xlsx-extractor@3.3.1
This release includes a patch update for the `@flatfile/plugin-xlsx-extractor` package. The patch fixes miscellaneous bugs and updates the `@flatfile/util-extractor` dependency. Specific changes include handling null values when detecting data rows and subheaders, and accounting for null cells when performing fuzzy matching between header and data rows. The package version has been updated from 3.3.0 to 3.3.1.
