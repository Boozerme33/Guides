@flatfile/plugin-job-handler@0.7.0
The release includes several updates to the @flatfile/plugin-job-handler package. It swaps the package's bundler from rollup to tsup. The package has been updated to use the latest version of the @flatfile/util-common dependency. The Jest configuration has been updated with environment settings, transforms, setup files, timeout, global setup, force exit, and pass with no tests options. The package.json file has been updated with new script commands for building, checking, linting, and testing the package. The main entry point has been changed to use the .js extension instead of .mjs. The browser field has been updated to use .js and .cjs extensions. The export map has been updated to define paths for different environments. The release note highlights that the developer using this package should be aware of these configuration and dependency updates.
