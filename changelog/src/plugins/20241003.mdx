### October 03, 2024
##transform

@flatfile/plugin-autocast@2.0.1
This release updates the @flatfile/autocast plugin to version 2.0.1. The @flatfile/api dependency has been updated to improve regional support. Additionally, the @flatfile/plugin-record-hook and @flatfile/util-common dependencies have been updated to their latest versions 1.7.1 and 1.4.1 respectively. Developers using this plugin should note that the updated @flatfile/api dependency may affect regional functionality.
