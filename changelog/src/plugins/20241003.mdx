@flatfile/plugin-export-workbook@1.0.1
This release updates the @flatfile/plugin-export-workbook package to version 1.0.1. The @flatfile/api dependency has been updated to improve regional support. The @flatfile/plugin-job-handler and @flatfile/util-common peer dependencies have also been updated to their latest versions, 0.6.1 and 1.4.1 respectively. There are no breaking changes or new features, but the updates should provide improved compatibility and stability.
