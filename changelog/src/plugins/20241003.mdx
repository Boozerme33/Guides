### October 03, 2024
##core_plugin

@flatfile/plugin-space-configure@0.6.1
This release updates the @flatfile/plugin-space-configure package to version 0.6.1. The main change is updating the @flatfile/api dependency to improve regional support. The peerDependencies @flatfile/api and @flatfile/listener have also been updated to versions 1.9.19 and 1.1.0 respectively. Additionally, the @flatfile/plugin-job-handler dependency has been updated to version 0.6.1, and the @flatfile/utils-testing devDependency has been updated to version 0.3.1.
