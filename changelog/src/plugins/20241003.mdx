### October 03, 2024
##automations

@flatfile/plugin-automap@0.5.1
This release updates the @flatfile/plugin-automap package to version 0.5.1. The @flatfile/api dependency has been updated to version 1.9.19, which improves regional support. Additionally, the @flatfile/listener peer dependency has been updated to version 1.1.0, and the @flatfile/utils-testing dev dependency has been updated to version 0.3.1.
