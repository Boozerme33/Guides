### October 29, 2024
## extractors

@flatfile/plugin-xml-extractor@0.8.0
The release includes several updates to the @flatfile/plugin-xml-extractor package. The package's bundler has been swapped to tsup. The @flatfile/util-extractor dependency has been updated to version 2.2.0. A Jest configuration file has been added to the package for running tests. The package now supports module and CommonJS exports for both Node.js and browser environments. The package scripts have been updated to use tsup for building and testing the package. The package structure has been updated to follow the modern npm package layout convention, with distribution files located in the dist directory. There are no changes to the public interface or functionality of the package itself.
