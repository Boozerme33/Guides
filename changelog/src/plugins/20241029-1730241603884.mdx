### October 29, 2024
## transform

@flatfile/plugin-dedupe@1.3.0
The release introduces several changes to the @flatfile/plugin-dedupe package. The package's bundler has been swapped to tsup for building the production bundle. The package's dependencies have been updated to @flatfile/plugin-job-handler@0.7.0 and @flatfile/util-common@1.5.0. A new jest.config.cjs file has been added for configuring Jest tests. The package.json file has been updated with new scripts for building, linting, and testing the package. The exports field has been updated to define the entry points for different environments. The package has been marked as a module with the "type": "module" field. The package's repository URL has been updated to include the Git protocol.
