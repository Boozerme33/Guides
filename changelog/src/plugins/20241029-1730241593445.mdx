### October 29, 2024
## core_plugin

@flatfile/plugin-constraints@4.0.0
The release notes for @flatfile/plugin-constraints v4.0.0 are as follows:

The bundler for the package has been swapped to tsup. The dependencies have been updated, including a new version of @flatfile/plugin-record-hook (v1.8.0). 

The package now supports Node.js 16 and above. Internally, the bulkRecordHookPlugin has been renamed to bulkRecordHook. The configuration for Jest has been updated, and new scripts have been added for linting, type checking, and testing.

There are no changes to the external interface or API for developers using this package. The updates are focused on internal tooling and dependencies.
