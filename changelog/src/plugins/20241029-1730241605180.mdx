### October 29, 2024
## extractors

@flatfile/plugin-foreign-db-extractor@0.3.0
The release introduces several changes to the @flatfile/plugin-foreign-db-extractor package. The bundler has been swapped from Rollup to tsup. A new jest.config.cjs file has been added for Jest configuration. The package.json file has been updated with new scripts, exports, and dependencies. The entry point for the package has been updated to use the .js extension instead of .mjs. The version has been bumped to 0.3.0, indicating a minor release with backwards-compatible changes.
