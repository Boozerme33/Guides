@flatfile/plugin-graphql-schema@1.3.0
This release updates the @flatfile/plugin-graphql-schema package to version 1.3.0. It updates several Flatfile dependencies to their latest versions, including @flatfile/plugin-space-configure which is updated to version 0.6.0. The peerDependency @flatfile/listener is also updated to version 1.0.5. Developers using this package should ensure they have the latest compatible versions of the dependencies installed.
