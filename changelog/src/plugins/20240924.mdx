@flatfile/plugin-convert-json-schema@0.4.0
This release updates the @flatfile/plugin-convert-json-schema package to version 0.4.0. The release contains a minor change where several Flatfile dependencies, including @flatfile/plugin-space-configure, were updated to their latest versions. There is also a patch change where the peer dependencies @flatfile/api and @flatfile/listener were updated to versions 1.9.13 and 1.0.5 respectively. Developers using this package should ensure they update their project dependencies accordingly.
