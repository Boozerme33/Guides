##automations

@flatfile/plugin-automap@0.5.0
This release updates the @flatfile/plugin-automap package to version 0.5.0. It updates several Flatfile dependencies to their latest versions, including @flatfile/listener which is now at version 1.0.5, and @flatfile/utils-testing which is now at version 0.3.0. Developers using this package should ensure their application is compatible with the updated dependency versions.
