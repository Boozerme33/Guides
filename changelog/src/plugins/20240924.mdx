##schema

@flatfile/plugin-convert-yaml-schema@0.3.0
This release updates the @flatfile/plugin-convert-yaml-schema package to version 0.3.0. It includes a minor update that brings several Flatfile dependencies to their latest versions. Additionally, there are patch updates for the dependencies @flatfile/plugin-space-configure (updated to 0.6.0) and @flatfile/plugin-convert-json-schema (updated to 0.4.0). The peer dependencies @flatfile/api and @flatfile/listener have also been updated to their latest versions, 1.9.13 and 1.0.5 respectively. The devDependency @flatfile/utils-testing has been updated to version 0.3.0.
