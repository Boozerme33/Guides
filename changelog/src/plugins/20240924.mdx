@flatfile/plugin-autocast@2.0.0
This release updates the @flatfile/plugin-autocast package to version 2.0.0. It includes a minor change that updates several Flatfile dependencies to their latest versions. Additionally, there are patch changes that update the @flatfile/plugin-record-hook dependency to version 1.7.0 and the @flatfile/util-common dependency to version 1.4.0. Consumers of this package should be aware that the updated dependencies may introduce changes or bug fixes that could affect their applications. The peer dependencies have also been updated, requiring @flatfile/listener version 1.0.5 or later and @flatfile/plugin-record-hook version 1.7.0 or later.
