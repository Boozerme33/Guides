### October 29, 2024
## extractors

@flatfile/plugin-zip-extractor@0.7.0
The zip extractor plugin has been updated to version 0.7.0. The bundler for the package has been swapped to tsup. The dependencies @flatfile/plugin-job-handler, @flatfile/util-file-buffer, and @flatfile/util-common have been updated to their latest versions. The package now supports ES module imports and exports in addition to CommonJS. The build scripts have been updated accordingly. Developers using this package should note the dependency updates and the new support for ES module imports.
