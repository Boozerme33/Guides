### October 29, 2024
## extractors

@flatfile/plugin-delimiter-extractor@2.3.0
The release updates the delimiter-extractor plugin to version 2.3.0. It swaps the package bundler from Parcel to tsup. The package dependencies have been updated, including upgrading @flatfile/util-extractor to version 2.2.0. The Jest configuration file has been renamed from jest.config.js to jest.config.cjs. The package.json file has been updated with new scripts for building, checking, linting, and testing the package using tsup and other tools. The export fields have been added to the package.json file, and the module distribution files have been updated accordingly. The parser.ts file has been updated with minor code changes related to importing and using the DelimiterOptions type.
