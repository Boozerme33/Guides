### September 26, 2024
##extractors

@flatfile/plugin-xlsx-extractor@3.2.0
This release introduces a new option `skipEmptyLines` that allows skipping empty lines when parsing XLSX files. When `skipEmptyLines` is set to `true`, any rows that contain only empty or whitespace values will be omitted from the parsed data. This can be useful for cleaning up the output and removing unnecessary blank rows. For example, if the input XLSX file contains:

```
column1, column2, column3
value1, value2, value3

value4, value5, value6

value7, value8, value9
```

With `skipEmptyLines: true`, the parsed output will be:

```
[
  { column1: 'value1', column2: 'value2', column3: 'value3' },
  { column1: 'value4', column2: 'value5', column3: 'value6' },
  { column1: 'value7', column2: 'value8', column3: 'value9' }
]
```

Without `skipEmptyLines` or with `skipEmptyLines: false`, the parsed output would include the empty row:

```
[
  { column1: 'value1', column2: 'value2', column3: 'value3' },
  {},
  { column1: 'value4', column2: 'value5', column3: 'value6' },
  {},
  { column1: 'value7', column2: 'value8', column3: 'value9' }
]
```
